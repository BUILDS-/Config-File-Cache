tex/   : random tex class files that must be in your tex-mf path
.vimrc : vim config
.vim/  : vim plugins and documentation

vim plugins I use:
a.vim           : sets :A to alternate between .c and .h files, etc
argtextobj.vim  : introduces 'argument' text object (a) for functions
bufexplorer.vim : better support for navigating buffers
fugitive.vim    : comprehensive git integration
minibufexpl.vim : lists all open files in a small top window
NERD_tree.vim   : shows directories in a sidebar
surround.vim    : introduces 'surround' motion (s)
tagbar.vim      : uses ctags to make navigating source files easier
tcomment.vim    : introduces comment motion (gc)
